# Key Features

## 4.5 Core Features

1.  **Dynamic Scoreboard**: Updates in real-time with visual indicators for the server.
2.  **Live Rankings Table**: Sortable columns for Wins, Losses, and Ranking with distinct up/down indicators.
3.  **Detailed Match View**: Deep dive into specific match stats (Aces, Double Faults) and player "Head-to-Head" comparisons.
4.  **Dark Mode Support**: Fully responsive dark/light theme switching.
5.  **Skeleton Loading**: Premium loading states (no basic spinners!) for a smoother UX.

## 6. Bonus Features

-   **Animated UI**: "Live" red dot indicators, smooth tab transitions, and row layout animations in the ranking table.
-   **Flag Integration**: Dynamic display of country flags for all players.
-   **Upcoming Match Mode**: Special "Tale of the Tape" view for scheduled matches.
